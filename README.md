# Информационные технологии и программирование (2 семестр)

Учебный репозиторий для студентов 1 курса (2 семестр) направления **«Прикладная математика и информатика»**. Основной фокус семестра — объектно-ориентированное программирование (ООП) на языке C# и глубокое понимание платформы .NET.

**Преподаватель:** Клюкин Даниил Анатольевич  
**Кафедра:** ПМиИТ  
**GitHub:** [DaniilKlyukin](https://github.com/DaniilKlyukin)

---

## 📚 Содержание лекций

| № | Тема | Ключевые концепции |
|:---:|:---|:---|
| 1 | **[Классы и структуры](Лекции/ИТиП%202%20лек%201.pptx)** | Конструкторы, `this`, цепочки вызовов, инициализаторы, `readonly`, `namespace`. |
| 2 | **[Инкапсуляция](Лекции/ИТиП%202%20лек%202.pptx)** | Модификаторы доступа (`private`, `protected`, `internal`), свойства (get/set), `init`, `required`. |
| 3 | **[Наследование и Полиморфизм](Лекции/ИТиП%202%20лек%203.pptx)** | Базовые/производные классы, `base`, `sealed`, приведение типов (`is`/`as`), `virtual`/`override`. |
| 4 | **[Продвинутые возможности](Лекции/ИТиП%202%20лек%204.pptx)** | Статические члены, перегрузка операторов, методы расширения (Extension), библиотеки классов (.dll). |
| 5 | **[Интерфейсы](Лекции/ИТиП%202%20лек%205.pptx)** | Контракты, множественная реализация, системные интерфейсы (`IEquatable`, `IComparable`, `ICloneable`). |
| 6 | **[Обобщения и Итераторы](Лекции/ИТиП%202%20лек%206.pptx)** | Generics (`List<T>`, `HashSet<T>`), ограничения `where`, интерфейсы `IEnumerable`/`IEnumerator`, `yield return`. |
| 7 | **[Память и ресурсы](Лекции/ИТиП%202%20лек%207.pptx)** | Стек vs Куча, упаковка/распаковка (boxing), Garbage Collector, `try-catch-finally`, `IDisposable`, `using`. |

## 🛠 Инструменты и требования
*   **Среда разработки:** Visual Studio Community.
*   **Платформа:** .NET 8.0 / .NET 9.0.
*   **Правила оформления:** 
    *   Все поля классов должны быть `private`.
    *   Доступ к данным — только через свойства или методы.
    *   Обязательное использование XML-комментариев для публичных методов.
